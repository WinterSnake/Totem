# Totem
A C library that incorporates Zig-like allocators into C.
Has a VTable interface for generic allocations and custom allocator implementations.

## Current Allocators
- [x] Fixed Buffer
- [ ] Page
- [ ] Brk/SBrk
- [ ] Linear
- [ ] Stack
- [ ] Arena
- [ ] Pool
- [ ] Slab
- [ ] Buddy
- [ ] General Purpose
